# 60-Duke
**60 DUKE: Ultimate Motion** to survivalowy body-horror RPG. Jako Duke, chronisz Mary w postapokaliptycznym bunkrze, walcząc z mutacją „Spremusa”. Faza zbierania 3D wykorzystuje **żyroskop i dotyk**, a system przetrwania oferuje zarządzanie zasobami, mroczne interakcje i syntezatorowe dźwięki. Nowoczesne, responsywne UI na PC i mobile. ☢️🥫🧬
